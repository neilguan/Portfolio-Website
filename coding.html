Description

An application built in Java with the JavaFX toolkit and the Selenium library. Generates a college schedule. It first fetches and parses course data from the University of Massachusetts at Amherst's web portal through web scraping. Again through web scraping, it fetches and parses course professor ratings from www.ratemyprofessor.com. It filters course data acquired from web scraping based on user input; for instance, the user can deselect 8 AM classes or classes that have mediocre professor ratings, and uses said data and filters to automatically generates college schedules. User may export schedule as a PNG file. Useful for college students that need to create their own schedule.

Description

An application that automates the harvesting of nether wart in Minecraft through a sequence of automated key inputs. Written in Autohotkey. After in-game creation of a specific type of farm as designated by specific instructions, user may run the macro. The macro will move the player in a specific pattern which automates the collection, harvesting, and replanting of nether wart. Useful in faction servers, wherein nether wart is sold to increase ingame currency. Use at own risk, as botting may be prohibited on many Minecraft servers.
